# Практична робота №10
## Множини. Set та HashSet в Java Collections Framework

У даній практичній роботі реалізовано систему логування подій на сайті з використанням множини `HashSet<User>`.

## Реалізовано:
- клас `User` з полями:
  - `id`
  - `name`
  - `password`
  - `lastLoginDate`
  - `isLoggedIn`
- перевизначення `equals()` та `hashCode()`
- клас `UserRegistry` для роботи з користувачами
- реєстрацію користувача
- вхід у систему
- вихід із системи
- перевірку реєстрації користувача
- видалення користувача
- підрахунок кількості унікальних користувачів
- виведення всіх користувачів
- консольне меню для демонстрації роботи програми

## Використані технології
- Java
- Set
- HashSet
- LocalDateTime
